
**Record** — это специальный тип класса в **Java**, который автоматически делает много работы за программиста. Он автоматически добавляет всё необходимое: поля, конструктор, методы для чтения данных, сравнение объектов и строковое представление.

**Пример:**
![[Pasted image 20250322001519.png]]
- **Person** — имя класса.
- **String name** и **int age** — это данные, которые будут храниться.

Когда пишешь такой код, Java сама создаёт:
- Приватные неизменяемые поля (*private final String name; и private final int age;*).
- Конструктор, который принимает **name** и **age** и задаёт их значения.
- Методы-геттеры: name() и age() (*без "get" в начале — это стиль Record*).
- Методы **equals()** и **hashCode()** для сравнения объектов.
- Метод **toString()** для красивого вывода.


**Пример использования:**
![[Pasted image 20250322001823.png]]
- Создаём объект с помощью конструктора **new Person("Маша", 25)**.
- Читаем данные через **name()** и **age()**.
- Метод **toString( )** автоматически выводит данные в читаемом виде.
- **equals( )** сравнивает два объекта по значениям полей.
